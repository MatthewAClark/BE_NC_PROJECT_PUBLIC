Personal Rail Travel App

Introduction

This is the backend API for the Personal Rail travel app, developed as a final project at Northcoders in Manchester. The purpose of the app is to build a database of train times and their running performance for a travellers individual needs. A passenger can look up only the train departures they are interested in,

The app uses real rail data provided by the UK's National Rail network through a third party API database, Transport API.
