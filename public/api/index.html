Personal Rail Travel API Page

The API is divided into two sections, live and db. The live section is used for fetching live API requests from the rail network for setting up new stations and routes. The db section is used for the stored train times, as well as performance statistics

Live

follows /api/live
GET /route/?from=station_code&to=station_code Fetch the live route between two stations
GET /service/:id Returns data for a particular train service
GET /station/route/?station_from=station_code&station_to=station_code&date=date_of_travel&time=time_of_travel&offset=hours find route and times between two stations on a particular date and time
GET stationtimes/:station_name get live station departures from a particular station
GET /station/:station_name Fetch station data

Schedules Database

follows /api/db
GET /schedules Fetches all train schedules
GET /schedules/?departure_time=time fetches a schedule at a specific departure time
GET /schedules/?departure_time_from=time&departure_time_to=time Fetches all departures between a range of departure times
GET /schedules/time/routes/?from=time&to=time Fetch schedules of a certain route between different times
GET /schedules/id/:schedule_id Fetch a schedule by it's ID
POST /schedules add a new schedule
GET /schedules/route/:route_id fetch schedules by a route id
GET /schedules/route/:route_id/?departure_time_from=time&departure_time_to=time fetch schedules by route id between different times
GET /schedules/all fetches all schedules stored in database
DELETE /schedules/:schedule_id delete a schedule by schedule id

Stations database

follows /api/db
GET /stations fetch all stations
GET /stations/?code=station_code Fetch a station by station code
GET /stations/:station_id Fetch station by station id
POST /stations add a new station
DELETE /staitons:station_id delete station by station id

Routes Database

follows /api/db
GET /routes fetchallroutes
POST /routes post a route
GET /routes/start fetchStartStation
GET /routes/start/:start_id fetch start station by start id
DELETE /routes/:route_id Delete route by ID

Train Performance Database

follows /api/db
GET /status Fetch all train performances
GET /status/schedules Fetch all performance data along with the schedules
GET /status/:status_id Fetch performance by ID
DELETE /status Delete a particular performance
POST /status Add new train performance data to the database
